# 回れ回れ

「回」という文字をひたすら回して成長させていく、シンプルなブラウザゲームです。  
回転数を増やすとレベルが上がり、機能が解放され、実績バッジも集められます。

公開ページはこちらです。  
https://mkunori.github.io/Kaiten/

## ゲームの目的

- 文字を回して総回転数を増やす
- レベルを上げる
- 新機能を解放する
- 実績バッジを集める

## 遊び方

1. 公開ページにアクセスします
2. 画面中央の `回` をクリックして回転数を増やします
3. 回転数が増えると、色変更・自動回転・エフェクト・覚醒モードなどの機能が順番に解放されます
4. `自動回転` は解放された瞬間に自動で `ON` になります
5. 実績バッジを集めながら、さらに回転数を伸ばしていきます

## ゲーム内の要素

### ステータスバー

- `総回転数`: 今までに回した回数
- `レベル`: 青天井で上がっていく現在レベル
- `次のレベルまで`: 今のレベル区間での進捗

### 中央ステージ

- 主役の `回` を大きく表示
- クリックで回転
- 解放状況や覚醒中の状態に応じて見た目が変化

### 下部の操作ボタン

- `自動回転`: 解放後にオン・オフできる
- `データリセット`: セーブデータを削除して最初からやり直す

### 解放要素

- 回転数を重ねると新しい機能が順番に解放されます
- 解放内容は事前に公開せず、遊びながら見つける形式です

### レベル

- レベルは上限なく上がります
- 次のレベルに必要な回転数は次の式で増えます
- `20 + (現在レベル - 1) × 10`
- 例
- `Lv1 → Lv2`: `20`
- `Lv2 → Lv3`: `30`
- `Lv3 → Lv4`: `40`

### 実績バッジ

総回転数実績とレベル到達実績があります。

総回転数実績:

- 円環の始まり: 最初の回転を刻んだ
- 微動の刻: わずかな揺らぎが動きへ変わる
- 流転の兆し: 回転は静かに流れを帯びはじめる
- 加速する円環: 円は意思を持つように勢いを増す
- 円環の探求者: 回転の理を追いはじめた者
- 共鳴する輪: 幾重もの回転が響き合う
- 流転の観測者: 絶えぬ動きを見つめ続ける者
- 永劫への接近: 終わりなき巡りへ手が届きはじめる
- 輪廻の担い手: 回り続ける力をその身に宿した
- 永久機関の証明: 回転はついに尽きぬ領域へ至る

レベル到達実績:

- 微光の目覚め: かすかな輝きが回転の奥で目を覚ます
- 流転の見習い: 巡りの理に触れはじめた者
- 円環の歩み手: 回転の道を確かな足取りで進みゆく
- 共鳴の担い手: 幾つもの巡りをその身に響かせる
- 永劫の探求者: 果てなき循環の深みを追い求める者
- 輪廻の継承者: 巡り続ける力を受け継ぎ、その先へ運ぶ
- 円環の証明者: 積み重ねた回転が到達を証明する

未獲得のバッジはロック状態で表示され、獲得すると見た目が変わります。
実績一覧はスクロール表示になっており、縦に伸びすぎないようにしています。

### 覚醒モード

- 総回転数 `100` で解放されます
- 解放されるまでは `？？？` として隠されています
- 解放後は右側パネルで状態を確認できます
- 通常時は覚醒ゲージを表示します
- 覚醒中は残り時間ゲージに切り替わります
- 覚醒ゲージは手動クリック時にだけ増えます
- 自動回転では覚醒ゲージは増えず、覚醒突入判定も行いません
- 覚醒中は回転時の加算量が `2倍` になります
- 覚醒名は以下からランダムで選ばれます
- `円環励起` `流転覚醒` `円環共鳴` `輪廻加速` `環動解放`
- 覚醒時間は `4秒〜10秒` のランダムです
- 覚醒終了後は `5秒` のクールタイムが入ります

## セーブ機能

- `localStorage` を使ってブラウザに自動保存します
- ページを再読み込みしても進行状況を復元します
- 保存対象は以下です
  - 総回転数
  - レベル
  - 解放済み機能
  - 実績獲得状態

- HTML

- JavaScript

## 開発について

このプロジェクトは Codex を活用しながら作成・調整しています。  
機能追加、UI改善、コメント追加、README整備などを段階的に進めています。

## ファイル構成

- `index.html`: 画面構造とUI要素
- `style.css`: ゲーム風UI、アニメーション、スクロール対応、レスポンシブ対応
- `script.js`: 回転処理、レベル計算、解放システム、実績、覚醒、演出、セーブ処理
