# 回れ回れ

「回」という文字をひたすら回して成長させていく、シンプルなブラウザゲームです。  
回転数を増やすとレベルが上がり、機能が解放され、実績バッジも集められます。

公開ページはこちらです。  
https://mkunori.github.io/Kaiten/

## ゲームの目的

- 文字を回して総回転数を増やす
- レベルを上げる
- 新機能を解放する
- 実績バッジを集める

## 遊び方

1. 公開ページにアクセスします
2. 画面中央の `回` をクリックして回転数を増やします
3. 下部の `回す` ボタンでも回転できます
4. 回転数が増えると、色変更・自動回転・エフェクトなどの機能が順番に解放されます
5. 右側の進行ゲージと実績バッジを見ながら、さらに回転数を伸ばしていきます

## ゲーム内の要素

### ステータスバー

- `総回転数`: 今までに回した回数
- `レベル`: 回転数に応じて上がるレベル
- `次の解放`: 次の機能解放までの残り回数

### 中央ステージ

- 主役の `回` を大きく表示
- クリックで回転
- 解放状況に応じて見た目が変化

### 下部の操作ボタン

- `回す`: 演出付きで1回回す
- `自動回転`: 解放後にオン・オフできる
- `データリセット`: セーブデータを削除して最初からやり直す

### 解放要素

- `10回`: 色変更
- `30回`: 自動回転
- `50回`: エフェクト

### 実績バッジ

以下のような条件で実績を獲得できます。

- 初回転
- 10回転達成
- 50回転達成
- 100回転達成
- 自動回転解放
- レベルアップ初心者

未獲得のバッジはロック状態で表示され、獲得すると見た目が変わります。

## セーブ機能

- `localStorage` を使ってブラウザに自動保存します
- ページを再読み込みしても進行状況を復元します
- 保存対象は以下です
  - 総回転数
  - レベル
  - 解放済み機能
  - 実績獲得状態
