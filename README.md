# 回れ回れ

シンプルな文字回転Webアプリです。  
画面中央の文字をクリックすると回転し、右側のカウンターで回転回数を確認できます。

## 機能

- 画面中央の文字をクリックすると、1回ごとに360度回転します
- 回転するたびに、右側のカウンターが1増えます
- ダブルクリックしても文字が選択されにくいように調整しています
- カウンター下部のボタンで、回転数を0にリセットできます
- 画面幅が狭い場合は、カウンターが下側に移動して文字と重なりにくくなります
- スマートフォン画面では、文字サイズや下部カウンターの幅も自動調整されます

## 画面構成

- 中央: クリックで回転する文字
- 右側: 回転数カウンター
- カウンター下部: リセットボタン

## 使用技術

- HTML
- CSS
- JavaScript

## 使い方

公開ページはこちらです。  
https://mkunori.github.io/Kaiten/

1. 上記URLにアクセスします
2. 画面中央の文字をクリックします
3. 文字が回転し、回転数がカウントされます
4. `0に戻す` ボタンを押すと、回転数と回転状態がリセットされます
